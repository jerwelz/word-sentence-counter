# Word and Sentence Counter [ENGLISCH]

A simple web application that receives a text input and displays:

- Word count  
- Sentence count  
- Average sentence length  

---

## Live Demo

https://calm-manatee-275846.netlify.app/


---

## Features

- Text input field
- Word and sentence analysis
- Average sentence length calculation
- Warning message for very long sentences
- Darkmode toggle
- Responsive mobile layout
- Language switch (English / German)

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM manipulation, event listeners)

---

## How to Use

1. Enter a text into the input field  
2. Click the **Analyze** button  
3. The results will appear below

## Wort- und Satzzähler [DEUTSCH]

Eine einfache Webanwendung, die eine Texteingabe empfängt und Folgendes anzeigt:

- Wortanzahl  
- Satzanzahl  
- Durchschnittliche Satzlänge  

---

## Funktionen

- Texteingabefeld
- Wort- und Satzanalyse
- Berechnung der durchschnittlichen Satzlänge
- Warnmeldung bei sehr langen Sätzen
- Umschaltung auf Dunkelmodus
- Responsives mobiles Layout
- Sprachumschaltung (Englisch / Deutsch)

---

## Verwendete Technologien

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM-Manipulation, Ereignis-Listener)

---

## Verwendung

1. Geben Sie einen Text in das Eingabefeld ein  
2. Klicken Sie auf die Schaltfläche **Analysieren**  
3. Die Ergebnisse werden unten angezeigt
